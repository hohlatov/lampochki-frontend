# ЛампоЗавод — Frontend

React-приложение интернет-магазина лампочек.

## Стек

- **React 18** + **Vite**
- **React Router DOM v6** — маршрутизация
- **CSS Modules** — стили
- Mock-данные 

## Страницы

| Путь | Описание |
|------|----------|
| `/` | Главная — герой, популярные товары, акции |
| `/catalog` | Каталог с фильтрами (цоколь, яркость, фирма, цена) |
| `/product/:id` | Карточка товара |
| `/cart` | Корзина |
| `/order` | Оформление заказа |
| `/confirmation` | Подтверждение заказа |

## Запуск

```bash
npm install
npm run dev
```

Открыть: http://localhost:5173

## Сборка

```bash
npm run build
```
